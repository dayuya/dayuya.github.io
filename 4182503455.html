esp01s+继电器可以不带光耦

实现手动开关 接线 3v3+io2

我研究了一下meplhaa脚本，把你的脚本改成这样：
{
“c”: {
“io”: [ [ [ 0, 1 ], 2 ], [ [ 2 ], 6 ] ],
“l”: 1, “b”: [ [ 0, 5 ] ]
},
“a”: [{
“t”: 1,
“s”: 0,
“0”: { “r”: [ [ 0, 1 ] ] },
“1”: { “r”: [ [ 0, 0 ] ] },
“b”: [ [ 2 ] ]
}]
}
这样的样子我的开关才成功完成动作。
